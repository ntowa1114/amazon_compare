# 🛒 Amazon商品比較Webアプリ

Amazonの商品URLを2つ入力することで、  
**価格・スペック・レビューを比較し、AIによる判断・アドバイスを提示するWebアプリ**です。

---

## 📌 開発背景
Amazonでは類似商品が多く、  
- スペックの違いが分かりにくい  
- 専門知識がないと正しい選択が難しい  

といった課題があります。

例えば、モバイルバッテリーでは  
「容量は同じでも出力（45W / 87W）によって価格が異なる」  
といったケースが多く、初心者にとって判断が困難です。

本アプリでは、**URLを貼るだけで比較結果と判断材料を可視化**し、  
購入判断を支援することを目的としています。

---

## 🚀 主な機能

### ① 商品URL入力
- Amazonの商品URLを2つ入力するだけで比較可能
- 対応ドメイン：amazon.co.jp / amazon.com

### ② 商品情報の取得・整理
- 商品名、価格、レビュー評価、スペック情報を取得
- 情報を整形し、比較表として表示

### ③ 商品比較画面
- 商品画像・価格・レビュー評価を横並び表示
- 各スペック項目を表形式で比較
- 優劣が明確な項目は色分け表示

### ④ AIによる判断・アドバイス
- スペック・価格・レビューをもとに総合判断
- 「どのような人に向いているか」を文章で提示

### ⑤ レビュー要約
- 大量のレビューから  
  利点・欠点を要約し、比較表示

## ⚙️ 処理の流れ（概要）

1. ユーザーが2つの商品URLを入力
2. 商品ページから情報を取得（スクレイピング）
3. 取得データを整理・整形
4. 比較表を生成
5. AI用プロンプトを作成
6. AIが比較結果を分析・要約
7. ユーザーへ結果を表示

---

## 💡 工夫した点

### 見るだけで分かるUI設計
- Amazonの商品ページを行き来せずに比較可能
- 横並びレイアウトによる直感的な比較体験

### 初心者目線の情報提供
- 専門用語や数値の意味を解説
- 実際の利用シーンを想像しやすいAI解説

### 情報の取捨選択
- 必要な情報を整理し、比較に不要な要素を排除
- 表と文章の役割を分離して可読性を向上

---

## ⚠️ 課題

- **取得データの誤認**
  - 商品ページの表記差により誤った情報を取得する可能性がある

- **AI判定の限界**
  - レビュー評価など主観的要素を数値的に優先しがち
  - 利用者の価値観を十分に反映できない場合がある
